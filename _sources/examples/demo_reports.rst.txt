HTML
----

Measurement Report
++++++++++++++++++

The following images show an HTML measurement report with

-   a custom Sphinx stylesheet/theme (based on the ReadTheDocs theme)
-   many components from the `sphinx-design <https://sphinx-design.readthedocs.io/en/latest/>`_ extension
    (included in `pharaoh-report`)
-   custom JS components to render Pandas dataframes as interactive tables using `DataTables <https://datatables.net/>`_
    (included in `pharaoh-report`)
-   time series plots created with Plotly
